# Telegram AI Userbot

Гібридний Telegram-бот та юзербот для читання, аналізу та відстеження діалогів за допомогою штучного інтелекту Gemini. 

Проект реалізований за допомогою бібліотек **Pyrogram** (для читання повідомлень від вашого імені) та **python-telegram-bot** (для інтерфейсу керування ботом).

## Основний функціонал

Бот взаємодіє з користувачем через наступні команди (англійською мовою):

1. **Діагностика системи:**
   * `/ping` — перевірка працездатності: тестує підключення вашого Telegram-бота, клієнта Pyrogram (юзербота) та сервісу Gemini AI.

2. **Аналіз чатів та діалогів:**
   * **Аналіз через повідомлення:** Надішліть боту ID чату та кількість повідомлень у форматі:
     ```text
     <chat_id>
     <msg_count>
     [опціонально: запит до AI]
     ```
     Бот завантажить останні повідомлення чату, збереже їх в JSON-файл і надішле вам разом з коротким підсумком від Gemini AI.
   * **Швидкі діалоги з AI:**
     * `/ai <запит>` — пряме звернення до Gemini AI із збереженням контексту розмови (до 20 повідомлень).
     * `/ai_clean` — очищення контексту діалогу з AI.
     * Відповідь на повідомлення бота (Reply) автоматично запускає Gemini AI для аналізу поточної історії чату.

3. **Списки діалогів:**
   * `/list <кількість> <тип>` — виводить останні діалоги. Підтримує фільтри типу чатів англійською та українською мовами:
     * `private`/`особисті`/`приватні` — особисті повідомлення.
     * `group`/`група`/`чати` — групи та супергрупи.
     * `channel`/`канал`/`тгк` — телеграм-канали.

4. **Відстеження статусів запрошень та домовленостей:**
   * `/track <години> <тема>` — сканує останні приватні чати, активні за вказану кількість годин, знаходить повідомлення, де ви когось запрошували або запитували про певну тему, та формує звіт про статус відповіді співрозмовника за допомогою AI:
     * ✅ **Agreed** (погодився)
     * ❌ **Declined** (відмовився)
     * ⏳ **Pending** (очікує відповіді / думає)
     * *Приклад:* `/track 12 день народження`

5. **Додаткові команди:**
   * `/id` — повертає ID повідомлення, на яке відповіли.
   * `/json` — надсилає тестовий JSON-файл.

## Стійкість до збоїв (Model Fallback)
У разі перевантаження серверів Google (помилка `503 UNAVAILABLE`) для моделі за замовчуванням `gemini-3.5-flash`, система автоматично переключиться на резервну модель `gemini-2.5-flash`, а за необхідності — на `gemini-2.0-flash`.

## Як запустити

1. **Встановлення залежностей**:
   ```bash
   pip install -r requirements.txt
   ```
2. **Створення файлу конфігурації**:
   Створіть файл `Python/config.py` з наступним вмістом:
   ```python
   admin_id = 565432599          # Ваш особистий Telegram ID
   TG_api_id = "2540621"         # ID з my.telegram.org
   TG_api_hash = "your_hash"     # Hash з my.telegram.org
   TGbot_token = "your_token"    # Токен бота від @BotFather
   AI_api_key = "your_gemini"    # Ключ Gemini з Google AI Studio
   ```
3. **Запуск**:
   ```bash
   python Python/tg-userbot.py
   ```
   *При першому запуску юзербот у консолі запитає ваш номер телефону та SMS-код від Telegram для створення сесії.*
